### WHAT'S NEW IN THIS VERSION

**Login Page**
- Redesigned to match school branding with school logo, motto "Per Ardua Ad Astra", split layout

**Dashboard**
- Removed: Fees Collected, Outstanding Fees
- Added: Total Boarding Students, Total Day Students, Total Teachers

**Students**
- Added: Boarding / Day student type field
- Added: Profile photo upload on enrollment
- Added: Delete student button (admin only)
- Students can login to their own portal
- Student portal: profile photo displayed, report card view, attendance view

**Teachers Portal (NEW)**
- Full teacher registration with profile photo
- Assign subjects to teachers per class
- Teachers can login with username/password
- Teacher dashboard shows assigned subjects
- Teachers can enter grades and mark attendance
- Teachers can view syllabus files

**Subjects (Updated to GES)**
Career Technology, Computing, Creative Arts and Design, English Language,
French, Ghanaian Language, History, Integrated Science, Mathematics,
Religious and Moral Education, Social Studies

**Mock Examination Portal – Basic 9 (NEW)**
- B.E.C.E grading system (1, 2, 3, 4, 5, 6, 7, 8, 9)
- Aggregate = Best 4 Core + Best 2 Elective subjects
- Core: English Language, Mathematics, Integrated Science, Social Studies
- Class results ranked by aggregate score
- Generate PDF class report

**Syllabus Portal (NEW)**
- Admin can upload and delete syllabus files (PDF, DOC, DOCX, PPT)
- Teachers can view and download uploaded files
- Filter by subject and class
